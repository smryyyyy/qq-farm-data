📊 效率说明

💰 每小时收益/经验

每小时收益：作物每小时的利润。利润 = 售价 - 种子成本。

每小时经验：作物每小时获得的经验值。经验用于提升农场等级。

每小时收益 = (售价 - 种子成本) × 季数 ÷ 总成熟时间

每小时经验 = 经验 × 季数 ÷ 总成熟时间

📈 总收益/经验

总收益：作物全部收获后的总利润（多季作物会累加）。

总经验：作物全部收获后获得的总经验。

总收益 = (售价 - 种子成本) × 季数

总经验 = 经验 × 季数

⏱️ 成熟时间

作物从播种到全部收获完成所需的时间。

多季作物的成熟时间是所有季数的总和，不同土地类型会影响成熟时间。

总成熟时间 = 首季时间 + 再熟时间 × (季数 - 1)

💡 土地加成

不同土地对作物有不同影响：

  • 🟫 普通土地：无加成
  • 🔴 红土地：增产+10%
  • ⬛ 黑土地：增产+20% · 成熟时间-20%
  • 🟡 金土地：增产+28% · 成熟时间-20%

🎯 使用建议

  • 想要快速赚钱：按"每小时收益"排序，选择前几名
  • 想要快速升级：按"每小时经验"排序，选择前几名
  • 想要最大化收益：按"总收益"排序，选择高价值作物
  • 低级玩家：先升级土地，再考虑高价值作物

❓ 常见问题

Q: Token 过期了怎么办？
A: Token 有效期为 30 天，过期后需要重新创建。可以在 GitHub 的 Settings → Developer settings → Personal access tokens 中查看和管理。
Q: 为什么收不到微信推送？
A: 请检查：1）SendKey 是否正确；2）是否点击了"测试推送"并成功；3）GitHub Actions 是否正常运行（查看仓库的 Actions 标签页）。
Q: 多设备数据会冲突吗？
A: 不会。每次同步时会合并数据，以最后修改时间为准。建议在不同设备上尽量避免同时编辑相同内容。
